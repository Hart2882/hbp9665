This repository is used to help develop Smart Contracts
 usuing an Account Abtraction ids to maintain the decentralization
 on the ETH blockchain.  

 Sometimes, due to price volatility, a bug, or a network error
 can cause errors with AA ids.

 **Smart contract is used as an entry point for ERC-4337 Account Abstraction 
        transactions.

Goal: To create a script or resource files to help with proper Smart Contracts and AA
        ids.  Using javascript or Visual Studio Basic Code you can create scripts to 
        help with the creation and stability of these Smart Contracts.
